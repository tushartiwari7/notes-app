# Notes App

> Stay Organized, Stay Creative
> Note Down your notes and boost your productivity.

- NOTE 
  - Create note
  - Delete Note
  - Edit Note
  - Change background color
  - Pin & UnPin
  - Filter & sort
- Trash, Delete & Aechive note lists.
- Authentication - LogIn, Signup.
